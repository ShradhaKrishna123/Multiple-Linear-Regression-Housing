# Multiple Linear Regression – Housing Dataset

This project demonstrates the implementation of **Multiple Linear Regression**
using Python on a housing dataset to predict house prices based on multiple features.

## Files in this Repository
- `Multiple_Linear_Regression.ipynb` – Jupyter Notebook containing data preprocessing, model building, and evaluation
- `housing.csv` – Dataset used for training and testing the model

## Steps Performed
- Data loading and exploration
- Handling missing values
- Feature selection
- Train-test split
- Model training using Multiple Linear Regression
- Model evaluation

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib / Seaborn
- Scikit-learn

## How to Run
1. Clone the repository
2. Install required libraries
3. Open the notebook and run all cells
